
  Шаг 1

  Программа предоставляет возможность:

  Списать книгу со склада (перевести в статус “отсутствует”)
  Создать заказ
  Отменить заказ
  Изменить статус заказа (новый, выполнен, отменен)
  Добавить книгу на склад (закрывает все запросы книги и меняет ее статус на “в
  наличии”)
  Оставить запрос на книгу



  ШАГ 2

  Программа должна позволять просматривать:

  Список книг (сортировать по алфавиту, дате издания, цене, наличию на складе)
  
  Список заказов (сортировать по дате исполнения, цене, статусу)
  
  Список выполненных заказов за период времени
  
  Сумму заработанных средств за период времени
  
  
  Список «залежавшихся» книг не проданы больше чем 6 мес
  
  Посмотреть детали заказа (какие-либо данные заказчика + книги)
  
  Посмотреть описание книги
